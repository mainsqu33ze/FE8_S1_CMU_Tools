# mainsqu33ze FE8 CMU HTML tools
Using this repository to host html web applications that I have built for use during my Fire Emblem 8 Create My Unit Ironman Hard mode run. 

## Unit creation randomizer
Use this to fully follow the rules defined in the [Mainsqu33ze FE8 CMU guide](https://docs.google.com/document/d/1HoySNDHte_pZ4KlqfQh1SshLDDM83BESv3rB_3BaHhQ/edit?tab=t.0#heading=h.b9tmdz4u684a) and make a unit in as simple as 3 button clicks.
* Select the unit to randomly replace
* Click randomize
* Click print
* Submit your unit on [Discord](https://discord.gg/ejZxsCMs5a)


## Boss randomizer
Use this to fully follow the rules defined in the [Mainsqu33ze FE8 CMU guide](https://docs.google.com/document/d/1HoySNDHte_pZ4KlqfQh1SshLDDM83BESv3rB_3BaHhQ/edit?tab=t.0#heading=h.b9tmdz4u684a) and make a boss unit in as simple as 3 button clicks.
* Select the unit to randomly replace
* Click randomize
* Click print
* [Video showing this off for the first 16 CMU bosses](https://youtube.com/shorts/nPdCJPduBh8?si=yUDgsgfRMbNMH24B)


## Combat tracker 
Use this to easily track key details during each chapter.
* Add the units in any order you would like to track.
*   The unit entered first will stay at the top of the list and the following units will populate the rows beneath them in sequential order.
* Simply click the buttons as the chapter unfolds.
* Click the print to csv button after each chapter to follow along with what I will be doing.
*   The data gathered from this will be used to populate an RShiny application.
